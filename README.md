# 🚀 Premium Time-Series Sensor Analytics Dashboard

A premium real-time sensor analytics dashboard built using Python, SQLite, and Streamlit.
This project simulates continuous sensor readings, stores timestamped data, and visualizes temperature analytics through a live interactive dashboard.

---

## 📌 Features

* Real-time sensor data generation
* Auto-refresh every second
* Time-series data storage using SQLite
* Premium glassmorphism dashboard UI
* Live temperature trend visualization
* Min / Max / Average temperature analytics
* Latest sensor records table

---

## 🛠️ Tech Stack

* Python
* Streamlit
* SQLite
* Pandas
* Matplotlib

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/kethabhargavi/Sensor.git
cd sensor_dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
streamlit run app.py
```

---

## 🌐 Dashboard Access

After running, open:

```text
http://localhost:8501
```
